▌一、深度神经网络基础词汇（242个）

单词/词组 词性 AI领域释义
artificial neural networkn.人工神经网络（ANN），简称神经网络
deep neural networkn.深度神经网络（DNN），有多个隐藏层的神经网络
perceptronn.感知机，最简单的神经网络，只有输入层和输出层
multi-layer perceptronn.多层感知机（MLP），有至少一个隐藏层的全连接神经网络
neuronn.神经元，神经网络的基本单元
connectionn.连接，神经元之间的连接，对应权重参数
activationn.激活值，神经元的输出值
activation functionn.激活函数，给神经网络引入非线性的函数
ReLUn.线性整流单元，最常用的激活函数，公式是max(0, x)
leaky ReLUn.带泄露的ReLU，解决ReLU的神经元死亡问题，x<0时输出ax而不是0
parametric ReLUn.参数化ReLU，a也是可学习的参数
randomized ReLUn.随机ReLU，训练时a是随机的，推理时固定
GELUn.高斯误差线性单元，大模型最常用的激活函数，公式是x*Φ(x)，Φ是高斯分布的累积分布函数
Swishn.Swish激活函数，公式是x*sigmoid(βx)，β是可学习参数
SiLUn.Sigmoid线性单元，Swish的特殊情况β=1
sigmoidn.S型激活函数，公式是1/(1+e^-x)，输出范围0到1，常用于二分类输出层
tanhn.双曲正切激活函数，公式是(e^x - e^-x)/(e^x + e^-x)，输出范围-1到1
softmaxn.Softmax激活函数，把输出转换成概率分布，总和为1，常用于多分类输出层
softplusn.Softplus激活函数，公式是ln(1+e^x)，是ReLU的平滑版本
mishn.Mish激活函数，公式是x*tanh(softplus(x))，性能比ReLU好
feedforward neural networkn.前馈神经网络，信号从输入层向前传到输出层，没有反馈连接
fully connected layern.全连接层，每个神经元和上一层所有神经元都连接
dense layern.全连接层的另一种叫法
linear layern.线性层，不带激活函数的全连接层
input layern.输入层，接收原始数据
hidden layern.隐藏层，输入层和输出层之间的层，其值在训练数据中没有标注
output layern.输出层，输出最终的预测结果
depthn.深度，神经网络的层数
widthn.宽度，神经网络每层的神经元数量
receptive fieldn.感受野，神经网络中某个神经元能看到的输入区域大小
forward propagationn.前向传播，输入数据从输入层经过各层计算得到输出的过程
backward propagationn.反向传播，根据损失函数计算梯度，从输出层往回更新各层参数的过程
chain rulen.链式法则，反向传播的数学基础，用来计算复合函数的导数
gradientn.梯度，损失函数对参数的导数，指示参数更新的方向
vanishing gradientn.梯度消失，梯度在反向传播过程中变得越来越小，导致浅层网络参数无法更新的问题
exploding gradientn.梯度爆炸，梯度在反向传播过程中变得越来越大，导致参数更新步长太大，训练不稳定的问题
gradient clippingn.梯度裁剪，把梯度的范数限制在某个阈值内，解决梯度爆炸问题
batch normalizationn.批量归一化（BN），在每一层的激活前对小批量数据做归一化，加速训练，缓解梯度消失
layer normalizationn.层归一化（LN），对每个样本的所有特征做归一化，适合NLP任务和循环神经网络
instance normalizationn.实例归一化（IN），对每个样本的每个通道单独做归一化，适合图像生成任务
group normalizationn.组归一化（GN），把通道分成组，在组内做归一化，适合小批量的情况
weight normalizationn.权重归一化，对权重参数做归一化，加速训练
residual connectionn.残差连接，把输入直接加到输出上，解决深度网络的退化问题，是ResNet的核心
skip connectionn.跳跃连接，残差连接的另一种叫法，也指U-Net中的编码器到解码器的连接
highway networkn. highway网络，引入门控机制的残差连接，允许信息直接通过
gated mechanismn.门控机制，控制信息流动的机制，比如LSTM中的门
dropoutn.随机失活，训练时随机让一部分神经元失活，防止过拟合的正则化技术
dropout raten.失活率，dropout中神经元失活的概率
dropconnectn.DropConnect，和dropout类似，但随机失活的是连接而不是神经元
SpatialDropoutn.空间dropout，随机失活整个特征图而不是单个神经元，适合卷积神经网络
DropPathn.路径丢弃，随机丢弃整个残差路径，常用在大模型中
stochastic depthn.随机深度，训练时随机丢弃一些层，本质和DropPath类似
weight decayn.权重衰减，L2正则化，在损失函数中加入权重的L2范数，防止过拟合
L1 regularizationn.L1正则化，在损失函数中加入权重的L1范数，能让权重变得稀疏
L2 regularizationn.L2正则化，即权重衰减
elastic net regularizationn.弹性网络正则化，同时使用L1和L2正则化
early stoppingn.早停，当验证集损失不再下降时提前停止训练，防止过拟合
data augmentationn.数据增强，通过对原始数据做变换来扩充数据集，防止过拟合，提升泛化能力
training setn.训练集，用来训练模型的数据集
validation setn.验证集，训练过程中用来评估模型性能，调整超参数的数据集
test setn.测试集，训练完成后用来最终评估模型泛化能力的数据集，不能用来调整超参数
batchn.批次，训练时一次输入的样本集合
batch sizen.批次大小，一个批次中的样本数量
mini-batchn.小批次，比较小的批次大小，是现在最常用的训练方式
epochn.轮次，把整个训练集全部训练一遍的过程
iterationn.迭代，一次参数更新的过程，即一个批次的前向传播加反向传播
optimizern.优化器，根据梯度更新参数的算法
Stochastic Gradient Descentn.随机梯度下降（SGD），最基础的优化算法，每次用一个样本的梯度更新参数
Mini-batch Gradient Descentn.小批量梯度下降，每次用一个批次的样本梯度更新参数，是最常用的SGD变体
Momentumn.动量，在SGD中加入之前的梯度的指数移动平均，加速收敛，减少震荡
Nesterov Accelerated Gradientn.NAG，涅斯捷罗夫加速梯度，动量的改进版，先更新参数再计算梯度
AdaGradn.自适应梯度算法，对每个参数自适应调整学习率，适合稀疏数据
RMSPropn.均方根传播，AdaGrad的改进版，用梯度平方的指数移动平均来调整学习率
Adadeltan.Adadelta，RMSProp的改进版，不需要手动设置学习率
Adamn.自适应矩估计，最常用的优化算法，结合了Momentum和RMSProp的优点
AdamWn.Adam的改进版，修正了权重衰减的实现，现在是大模型训练的标准优化器
Adamaxn.Adam的变体，基于无穷范数，更稳定
Nadamn.结合了NAG和Adam的优化算法
Lionn.最近提出的优化算法，比AdamW更省内存，性能更好，大模型训练中逐渐开始使用
learning raten.学习率，参数更新的步长，是最重要的超参数之一
learning rate schedulingn.学习率调度，训练过程中动态调整学习率的策略
step decayn.步长衰减，每隔几个epoch把学习率乘以一个衰减因子
exponential decayn.指数衰减，学习率随训练步数指数下降
cosine annealingn.余弦退火，学习率按照余弦函数的形式下降，效果很好，常用在大模型训练中
cosine annealing with warm restartsn.带热重启的余弦退火，学习率下降到最低点后会重新升高，再下降，能跳出局部最优
warmupn.学习率预热，训练刚开始的时候学习率从0慢慢升高到初始学习率，防止训练初期不稳定
linear warmupn.线性预热，学习率线性升高
constant warmupn.固定预热，前几步学习率固定
learning rate findern.学习率查找器，通过训练一小段时间观察不同学习率下的损失，找到最合适的初始学习率
loss functionn.损失函数，衡量模型预测结果和真实标签之间差距的函数
cost functionn.代价函数，整个训练集上的平均损失
objective functionn.目标函数，模型要最小化或者最大化的函数
Mean Squared Errorn.均方误差（MSE），回归任务最常用的损失函数，预测值和真实值差的平方的平均值
Mean Absolute Errorn.平均绝对误差（MAE），预测值和真实值差的绝对值的平均值，对异常点更鲁棒
Huber lossn.Huber损失，结合了MSE和MAE的优点，误差小的时候是MSE，误差大的时候是MAE，对异常点鲁棒
Cross Entropy lossn.交叉熵损失，分类任务最常用的损失函数
Binary Cross Entropyn.二分类交叉熵（BCE），二分类任务的损失函数
Categorical Cross Entropyn.多分类交叉熵，多分类任务的损失函数，标签是one-hot编码
Sparse Categorical Cross Entropyn.稀疏多分类交叉熵，多分类任务的损失函数，标签是整数编码，不需要one-hot
Focal lossn.焦点损失，交叉熵的改进版，降低易分类样本的权重，重点关注难分类样本，解决类别不平衡问题
Dice lossn.Dice损失，基于Dice系数的损失函数，常用在图像分割任务中，对类别不平衡鲁棒
IoU lossn.交并比损失，基于IoU的损失函数，常用在目标检测和分割任务中
Contrastive lossn.对比损失，自监督学习中常用的损失函数，让相似样本的表示更近，不相似样本的表示更远
Triplet lossn.三元组损失，对比损失的一种，由锚点、正样本、负样本组成，让锚点和正样本的距离小于锚点和负样本的距离
Margin lossn.间隔损失，常用在人脸识别等任务中
KL divergencen.KL散度，衡量两个概率分布之间的差异，常用在生成模型和知识蒸馏中
JS divergencen.JS散度，KL散度的对称版本
Wasserstein distancen.推土机距离（W距离），衡量两个概率分布之间的距离，比KL/JS散度更适合生成模型，是WGAN的基础
perplexityn.困惑度，语言模型的评估指标，等于交叉熵的指数形式，越低越好
metricn.评估指标，用来衡量模型性能的指标
accuracyn.准确率，分类正确的样本占总样本的比例
precisionn.精确率，预测为正的样本中真实为正的比例
recalln.召回率，真实为正的样本中被预测为正的比例
F1 scoren.F1分数，精确率和召回率的调和平均，综合衡量分类性能
ROC curven.ROC曲线，以假阳性率为x轴，真阳性率为y轴的曲线，衡量二分类模型的性能
AUCn.ROC曲线下的面积，取值0到1，越大越好
confusion matrixn.混淆矩阵，分类任务中真实标签和预测标签的对应矩阵
true positiven.真正例（TP），真实为正，预测也为正的样本
true negativen.真负例（TN），真实为负，预测也为负的样本
false positiven.假正例（FP），真实为负，预测为正的样本，也叫误报
false negativen.假负例（FN），真实为正，预测为负的样本，也叫漏报
true positive raten.真阳性率（TPR），即召回率
false positive raten.假阳性率（FPR），真实为负的样本中被预测为正的比例
mean Average Precisionn.平均精度均值（mAP），目标检测、信息检索等任务的常用指标
Intersection over Unionn.交并比（IoU），两个框的交集和并集的比值，目标检测和分割任务的常用指标
Dice coefficientn.Dice系数，和IoU类似，衡量两个集合的相似性，常用在分割任务中
Root Mean Squared Errorn.均方根误差（RMSE），MSE的平方根，回归任务的常用指标
Mean Absolute Percentage Errorn.平均绝对百分比误差（MAPE），相对误差，回归任务的常用指标
R-squaredn.R²，决定系数，衡量回归模型拟合程度的指标，取值0到1，越大越好
BLEU scoren.BLEU分数，机器翻译任务的常用评估指标，衡量生成译文和参考译文的相似度
ROUGE scoren.ROUGE分数，文本摘要任务的常用评估指标
perplexityn.困惑度，语言模型的常用评估指标，越低越好
top-k accuracyn.top-k准确率，预测的前k个结果中包含正确标签的样本比例
transfer learningn.迁移学习，把在一个任务上学到的知识迁移到另一个相关任务上，减少训练数据需求，提升性能
pre-trained modeln.预训练模型，在大规模数据集上预先训练好的模型，可以直接用或者微调
fine-tuningv./n.微调，在预训练模型的基础上，用目标任务的小数据集继续训练，调整参数适配目标任务
feature extractionn.特征提取，冻结预训练模型的参数，只用预训练模型提取特征，然后训练一个新的分类器
domain adaptationn.领域自适应，解决源领域和目标领域数据分布不一致的问题，让模型在目标领域也能有好的性能
domain shiftn.领域偏移，源领域和目标领域数据分布不一致的问题
few-shot learningn.小样本学习，只用很少的标注样本就能训练出好的模型
one-shot learningn.单样本学习，只用一个标注样本就能训练模型
zero-shot learningn.零样本学习，不需要目标任务的标注样本就能让模型完成任务
prompt learningn.提示学习，大模型时代的新学习范式，通过设计合适的提示来引导大模型完成任务，不需要微调或者只需要很少的参数调整
soft promptn.软提示，可学习的提示向量，不需要修改模型参数，只需要调整提示向量
hard promptn.硬提示，自然语言形式的提示，比如给模型的问题描述
prefix tuningn.前缀微调，只调整Transformer每层的前缀向量，其他参数冻结
LoRAn.低秩适配，大模型高效微调技术，只调整注意力层的低秩矩阵，参数效率很高
parameter efficient fine-tuningn.参数高效微调（PEFT），只调整很少的参数就能完成微调的技术，比如LoRA、前缀微调等

convolutionn.卷积操作，CNN的核心操作，用卷积核在输入上滑动计算得到特征图
convolutional kerneln.卷积核/滤波器，CNN的参数，用来提取特征
filtern.滤波器，即卷积核
