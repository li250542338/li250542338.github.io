🐍 Python 列表学习总结

List - 有序可变的万能容器

📖 什么是列表？

列表是 Python 中最常用的数据结构之一，它是一个有序、可变的元素集合。你可以把它想象成一个万能的购物车，什么都能往里放！

💡 核心特点

  • 有序：元素有固定的顺序，通过索引访问
  • 可变：可以随时添加、删除、修改元素
  • 异构：可以包含不同类型的元素（数字、字符串、甚至其他列表）
  • 索引从0开始：第一个元素的索引是 0

🛠️ 基本操作

1. 创建列表

# 方式一：使用方括号 [] numbers = [1, 2, 3, 4, 5] fruits = ["苹果", "香蕉", "橙子"] mixed = [1, "hello", 3.14, True] # 混合类型 # 方式二：使用 list() 函数 chars = list("hello") # ['h', 'e', 'l', 'l', 'o'] # 方式三：空列表 empty_list = [] empty_list = list() # 方式四：列表乘法（快速创建重复元素） zeros = [0] * 5 # [0, 0, 0, 0, 0]

2. 访问元素

fruits = ["苹果", "香蕉", "橙子", "葡萄"] # 正索引（从0开始） print(fruits[0]) # 输出: 苹果 print(fruits[2]) # 输出: 橙子 # 负索引（从-1开始，从右往左数） print(fruits[-1]) # 输出: 葡萄 print(fruits[-2]) # 输出: 橙子 # 切片（获取子列表） print(fruits[1:3]) # 输出: ['香蕉', '橙子'] print(fruits[:2]) # 输出: ['苹果', '香蕉'] print(fruits[2:]) # 输出: ['橙子', '葡萄'] print(fruits[::2]) # 步长为2，输出: ['苹果', '橙子'] print(fruits[::-1]) # 反转列表，输出: ['葡萄', '橙子', '香蕉', '苹果']

3. 添加元素

fruits = ["苹果", "香蕉"] # 在末尾添加单个元素 fruits.append("橙子") print(fruits) # 输出: ['苹果', '香蕉', '橙子'] # 在指定位置插入元素 fruits.insert(1, "葡萄") print(fruits) # 输出: ['苹果', '葡萄', '香蕉', '橙子'] # 合并另一个列表（在末尾扩展） more_fruits = ["西瓜", "芒果"] fruits.extend(more_fruits) print(fruits) # 输出: ['苹果', '葡萄', '香蕉', '橙子', '西瓜', '芒果'] # 使用 + 运算符合并（返回新列表） all_fruits = fruits + ["草莓"]

4. 修改元素

fruits = ["苹果", "香蕉", "橙子"] # 修改单个元素 fruits[1] = "葡萄" print(fruits) # 输出: ['苹果', '葡萄', '橙子'] # 修改多个元素（切片赋值） fruits[1:3] = ["芒果", "西瓜"] print(fruits) # 输出: ['苹果', '芒果', '西瓜']

5. 删除元素

fruits = ["苹果", "香蕉", "橙子", "香蕉"] # 删除指定位置的元素（并返回它） popped = fruits.pop(1) print(fruits) # 输出: ['苹果', '橙子', '香蕉'] print(popped) # 输出: 香蕉 # 删除第一个匹配的元素 fruits.remove("香蕉") print(fruits) # 输出: ['苹果', '橙子'] # 删除指定位置或切片 del fruits[0] # del fruits[1:3] # 清空整个列表 fruits.clear() print(fruits) # 输出: []

🔍 常用方法

📊 len()

获取列表长度

len(fruits)

🔍 index()

查找元素索引

fruits.index("苹果")

🔢 count()

统计元素出现次数

fruits.count("苹果")

🔀 sort()

原地排序

fruits.sort()

🔀 sorted()

返回新的排序后的列表

sorted(fruits)

🔄 reverse()

原地反转

fruits.reverse()

遍历列表

fruits = ["苹果", "香蕉", "橙子"] # 方式一：遍历元素（最常用） for fruit in fruits: print(fruit) # 方式二：遍历索引 for i in range(len(fruits)): print(f"{i}: {fruits[i]}") # 方式三：同时获取索引和元素（推荐） for idx, fruit in enumerate(fruits): print(f"{idx}: {fruit}")

✨ 进阶技巧

1. 列表推导式

# 创建一个平方数列表 squares = [x**2 for x in range(1, 6)] print(squares) # 输出: [1, 4, 9, 16, 25] # 带条件的列表推导式 numbers = [1, 2, 3, 4, 5, 6] evens = [x for x in numbers if x % 2 == 0] print(evens) # 输出: [2, 4, 6] # 转换元素 words = ["hello", "world"] upper_words = [word.upper() for word in words] print(upper_words) # 输出: ['HELLO', 'WORLD']

2. 嵌套列表（二维列表）

# 创建一个 3x3 的矩阵 matrix = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ] # 访问元素 print(matrix[0][0]) # 输出: 1 print(matrix[1][2]) # 输出: 6 # 遍历二维列表 for row in matrix: for num in row: print(num, end=" ") print()

3. 列表解包

# 基本解包 a, b, c = [1, 2, 3] print(a) # 输出: 1 # 用 * 收集剩余元素 first, *rest = [1, 2, 3, 4, 5] print(first) # 输出: 1 print(rest) # 输出: [2, 3, 4, 5] # 交换变量（不用临时变量！） a, b = 1, 2 a, b = b, a print(a, b) # 输出: 2 1

🏥 医院质控办实战示例

# 用列表管理科室质控检查记录 quality_records = [ {"科室": "内科", "日期": "2024-04-01", "合格率": 98.5}, {"科室": "外科", "日期": "2024-04-02", "合格率": 97.2}, {"科室": "儿科", "日期": "2024-04-03", "合格率": 99.1} ] # 计算平均合格率 total = sum(record["合格率"] for record in quality_records) average = total / len(quality_records) print(f"平均合格率: {average:.1f}%")

⚠️ 常见陷阱

❌ 索引越界

访问不存在的索引会报错 IndexError，注意列表长度！

❌ 列表是可变对象

赋值只是引用，修改新列表会影响原列表！用 copy() 或切片！

❌ 遍历中删除

不要在遍历列表时直接删除元素，会出问题！

如何正确复制列表

original = [1, 2, 3] # 错误方式：引用同一个对象 wrong_copy = original wrong_copy.append(4) print(original) # 输出: [1, 2, 3, 4] — 原列表也被修改了！ # 正确方式一：使用 copy() 方法 copy1 = original.copy() # 正确方式二：使用切片 copy2 = original[:] # 正确方式三：使用 list() 函数 copy3 = list(original)
