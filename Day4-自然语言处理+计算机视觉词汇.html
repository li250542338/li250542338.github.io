单词/词组 词性 AI领域释义
Natural Language Processingn.自然语言处理（NLP），让计算机理解和处理人类语言的技术
Computational Linguisticsn.计算语言学，NLP的理论基础
corpusn.语料库，NLP训练用的文本数据集
tokenn. token，文本切分后的基本单元，比如词、子词、字
tokenizationn.分词/切词，把文本切分成token的过程
word tokenizationn.词级别分词，把文本切分成词
subword tokenizationn.子词级别分词，把词切分成更小的子词单元，大模型常用
Byte-Pair Encodingn.字节对编码（BPE），最常用的子词分词算法，GPT系列用的就是BPE
WordPiecen.Google提出的子词分词算法，BERT系列用的是WordPiece
Unigram Language Modeln.一元语言模型分词算法，SentencePiece用的就是这个
SentencePiecen.Google开源的无监督分词工具，不需要预先分词，直接处理原始文本
vocabularyn.词表，所有token的集合
OOVn.未登录词，不在词表中的词
embeddingn.嵌入，把离散的token转换成连续的低维向量表示
word embeddingn.词嵌入，词的向量表示
Word2Vecn.经典的词嵌入算法，有CBOW和Skip-gram两种模式
CBOWn.连续词袋模型，用上下文词预测中心词
Skip-gramn.跳字模型，用中心词预测上下文词
GloVen.全局向量词嵌入，基于共现矩阵的词嵌入算法
FastTextn.Facebook提出的词嵌入算法，考虑了词的子词结构，能处理OOV词
positional encodingn.位置编码，Transformer中给token加入位置信息的方法，因为Transformer本身没有时序性
positional embeddingn.位置嵌入，可学习的位置表示，也是给token加入位置信息的方法
segment embeddingn.段落嵌入，BERT中用来区分两个句子的嵌入
language modeln.语言模型，计算句子概率或者预测下一个词的模型
autoregressive language modeln.自回归语言模型，从左到右逐个生成词，GPT系列就是自回归语言模型
autoencoding language modeln.自编码语言模型，预测被掩码的词，BERT就是自编码语言模型
encoder-decoder language modeln.编码解码语言模型，有编码器和解码器，适合生成任务，比如T5、BART
perplexityn.困惑度，衡量语言模型好坏的指标，越低越好
N-gramn.N元语法，传统的语言模型，下一个词的概率只依赖前面的N-1个词
unigramn.一元语法，N=1
bigramn.二元语法，N=2
trigramn.三元语法，N=3
smoothingn.平滑，解决N-gram模型中零概率问题的技术
Recurrent Neural Networkn.循环神经网络（RNN），适合处理序列数据的神经网络，有循环连接，能保留历史信息
Long Short-Term Memoryn.长短期记忆网络（LSTM），RNN的改进版，解决了RNN的长程依赖问题
Gated Recurrent Unitn.门控循环单元（GRU），LSTM的简化版，参数更少，速度更快
hidden staten.隐藏状态，RNN/LSTM/GRU中保存历史信息的状态
cell staten.细胞状态，LSTM中专门用来保存长期信息的状态
input gaten.输入门，LSTM中控制哪些信息进入细胞状态的门
forget gaten.遗忘门，LSTM中控制哪些信息从细胞状态中丢弃的门
output gaten.输出门，LSTM中控制哪些信息从细胞状态输出到隐藏状态的门
update gaten.更新门，GRU中的门，对应LSTM的输入门和遗忘门
reset gaten.重置门，GRU中的门，控制历史信息的重置
bidirectional RNNn.双向RNN，同时用从左到右和从右到左的RNN，能同时利用上下文信息
BiLSTMn.双向LSTM，最常用的双向循环神经网络
seq2seqn.序列到序列模型，输入是一个序列，输出也是一个序列，适合翻译、摘要等任务
encodern.编码器，seq2seq中处理输入序列，编码成上下文向量的部分
decodern.解码器，seq2seq中根据编码器的输出生成输出序列的部分
attention mechanismn.注意力机制，让解码器在生成每个词的时候关注输入序列中不同的位置，解决长序列信息丢失问题
self-attentionn.自注意力机制，注意力机制的一种，输入序列自己和自己做注意力，能捕捉序列内部的依赖关系
scaled dot-product attentionn.缩放点积注意力，Transformer中用的注意力机制，公式是Attention(Q,K,V) = softmax(QK^T/√d_k)V
queryn.查询（Q），注意力机制中的三个输入之一
keyn.键（K），注意力机制中的三个输入之一
valuen.值（V），注意力机制中的三个输入之一
attention weightn.注意力权重，softmax的输出，每个位置的权重代表对该位置的关注程度
multi-head attentionn.多头注意力，把Q,K,V分成多个头，每个头学习不同的注意力模式，然后拼接起来
Transformern.Transformer架构，完全基于自注意力机制，没有循环和卷积，现在是NLP和CV的主流架构
Transformer encodern.Transformer编码器，由多个编码器层堆叠而成，每层有多头自注意力和前馈网络两个子层
Transformer decodern.Transformer解码器，由多个解码器层堆叠而成，每层有掩码多头自注意力、交叉注意力和前馈网络三个子层
masked multi-head attentionn.掩码多头注意力，解码器用的自注意力，掩码掉当前位置后面的词，防止看到未来的信息
cross attentionn.交叉注意力，解码器中的注意力层，Q来自解码器的输出，K和V来自编码器的输出
feed-forward networkn.前馈网络，Transformer中的子层，由两个线性层中间加一个激活函数组成
residual connectionn.残差连接，Transformer中每个子层都有残差连接，然后接层归一化
layer normalizationn.层归一化，Transformer中每个残差连接后都做层归一化
BERTn.双向编码器表示，Google提出的预训练语言模型，开启了NLP的预训练时代
masked language modeln.掩码语言模型（MLM），BERT的预训练任务，随机掩码掉一些词，让模型预测被掩码的词
next sentence predictionn.下一句预测（NSP），BERT的预训练任务，判断两个句子是否是连续的
