1.3 腰围评估（中心性肥胖）

腰围≥同年龄、同性别儿童第90百分位，提示中心性肥胖

腰围测量方法：
受试者站立位，双足分开25-30cm，在腋中线肋弓下缘与髂嵴上缘连线中点水平，用软尺水平环绕腹部一周读取数值。

二、评估内容

2.1 详细病史采集

评估项目 具体内容
生长发育史 出生体重、身长；婴儿期喂养方式；各年龄段体重增长曲线；性发育情况
饮食史 每日进餐次数、时间；食物种类偏好；零食、饮料摄入情况；是否有暴饮暴食、夜间进食习惯
运动习惯 每日体力活动时间；运动类型；静态行为时间（看电视、玩手机、电脑）
家族史 父母及兄弟姐妹BMI；家族肥胖史；2型糖尿病史；高血压史；心血管疾病史
睡眠情况 每日睡眠时间；是否有打鼾、睡眠呼吸暂停；是否有白天嗜睡
心理社会因素 学习压力；同伴关系；是否有自卑、焦虑、抑郁等情绪问题；是否有因体重受到嘲笑

2.2 全面体格检查

  • 基本测量：身高、体重、BMI、腰围、臀围、腰臀比
  • 生命体征：血压（建议测双侧）、心率、呼吸频率
  • 皮肤检查：黑棘皮征（颈部、腋窝、腹股沟）、紫纹、多毛
  • 腹部检查：肝肿大、脂肪肝相关体征
  • 性发育评估：Tanner分期
  • 其他：甲状腺检查、关节检查、步态检查

  • 性激素（睾酮、雌二醇、LH、FSH）
  • 皮质醇节律
  • 肝脏超声
  • 心电图
  • 睡眠监测（怀疑睡眠呼吸暂停者）

2.4 并发症评估

  • 代谢综合征
  • 非酒精性脂肪肝
  • 高血压
  • 糖尿病前期/2型糖尿病
  • 血脂异常
  • 阻塞性睡眠呼吸暂停
  • 多囊卵巢综合征（女性）
  • 骨关节疾病
  • 心理社会问题

三、治疗方案

🎯 核心理念：
儿童肥胖治疗的目标是控制体重增长速度，而非快速减重！
通过生活方式干预，使身高增长、BMI逐渐下降至正常范围。

3.1 治疗原则

  1. 以生活方式干预为主：饮食调整、运动增加、行为改变是基础
  2. 家庭参与至关重要：家长起榜样作用，全家共同改变生活方式
  3. 循序渐进，持之以恒：每周减重不宜超过0.5-1kg
  4. 个体化治疗：根据患儿年龄、肥胖程度、并发症制定方案
  5. 不推荐药物作为一线治疗：仅用于重度肥胖且生活方式干预无效者

3.2 分阶段治疗策略

肥胖程度 初始治疗 强化治疗
超重 预防指导、生活方式干预 如6个月无改善，进入下一阶段
轻度肥胖 结构化生活方式干预 如3-6个月无改善，考虑药物
中度肥胖 强化生活方式干预 如3个月无改善，考虑药物
重度肥胖 多学科综合干预 尽早考虑药物，必要时手术

3.3 运动干预方案

运动目标

  • 有氧运动：每周5-7天，每天累计60分钟中等强度运动
  • 力量训练：每周2-3次，每次20-30分钟
  • 减少静态行为：每天屏幕时间≤2小时

运动强度评估

强度 主观感受 心率范围 运动类型
低强度 轻松，能正常交谈 最大心率的50-60% 慢走、做家务
中等强度 有点累，还能交谈 最大心率的60-70% 快走、慢跑、骑车、游泳
高强度 很累，难以交谈 最大心率的70-85% 快跑、跳绳、球类运动
最大心率估算公式：
最大心率 = 220 - 年龄（岁）

推荐运动类型

快走 慢跑 游泳 骑自行车 跳绳 篮球 足球 乒乓球 羽毛球 跳舞 武术 瑜伽

力量训练建议

  • 上肢力量：俯卧撑、哑铃、弹力带
  • 下肢力量：深蹲、弓步、跳台阶
  • 核心力量：仰卧起坐、平板支撑、卷腹
  • 频率：每周2-3次，每次2-3组，每组10-15次

3.4 行为干预方案

建立健康作息

年龄段 推荐睡眠时间 建议入睡时间
小学生（6-12岁）10小时21:00前
初中生（13-15岁）9小时22:00前
高中生（16-18岁）8小时22:30前

饮食行为技巧

  • 细嚼慢咽，每口食物咀嚼20-30次
  • 避免边吃边看电视、玩手机
  • 使用较小的餐盘和碗
  • 先吃蔬菜，再吃肉，最后吃主食
  • 记录饮食日记，包括食物种类、数量、进食时间
  • 定期监测体重，每周1-2次，固定时间

目标设定与奖励机制

  • 设定具体、可测量的小目标（如"每周运动5天"）
  • 目标达成后给予非食物奖励（看电影、买书籍、户外活动等）
  • 避免用食物作为奖励
  • 定期回顾目标，及时调整

3.5 药物治疗

⚠️ 药物治疗适应症：
1. BMI≥同年龄同性别第99百分位
2. 已进行至少6个月强化生活方式干预但效果不佳
3. 合并肥胖相关并发症（糖尿病、高血压、睡眠呼吸暂停等）

不推荐用于12岁以下儿童！

3.6 手术治疗

仅用于极端情况：青少年重度肥胖，且其他治疗无效，存在严重并发症者。需严格评估，由多学科团队共同决定。

4.5 实用饮食技巧

  • 餐盘分区法：一半蔬菜，四分之一蛋白质，四分之一主食
  • 细嚼慢咽：每口咀嚼20-30次，每餐20-30分钟
  • 小餐具策略：使用较小的餐盘和碗，视觉上感觉食物更多
  • 先进食顺序：先喝汤 → 再吃蔬菜 → 再吃肉 → 最后吃主食
  • 三餐规律：定时定量，不吃夜宵，避免过度饥饿后暴饮暴食
  • 减少在外就餐：外卖和餐馆通常油脂和盐分较高
  • 学会看食品标签：注意能量、脂肪、糖、钠含量
  • 家庭共同参与：全家一起改变饮食习惯，营造健康饮食环境

4.6 常见误区提醒

❌ 常见误区：
• 误区1：儿童快速减重没问题 → 可能影响生长发育，每周减重不宜超过0.5-1kg
• 误区2：完全不吃主食 → 主食是必需能量来源，选择全谷物即可
• 误区3：用水果代餐 → 水果含糖，不能完全替代正餐
• 误区4：只吃蔬菜不吃肉 → 蛋白质对生长发育很重要
• 误区5：喝果汁代替喝水 → 果汁含糖高，直接吃水果更好
• 误区6：用食物奖励孩子 → 会形成不健康的食物关联

五、随访与监测方案

5.1 随访频率

肥胖程度 随访频率 监测内容
超重 每3个月1次 体重、身高、BMI、饮食运动指导
轻度肥胖 每1-2个月1次 体重、身高、BMI、腰围、血压
中度肥胖 每月1次 上述 + 血糖、血脂、肝功能
重度肥胖 每2周1次 上述 + 并发症评估、多学科会诊

5.2 监测指标

体格测量

  • 体重：每周测量1-2次，固定时间（晨起空腹排便后）
  • 身高：每月测量1次
  • BMI：每次随访计算
  • 腰围：每月测量1次
  • 血压：每次随访测量

血脂✓每6个月每3个月
肝功能✓每6个月每3个月
肝脏超声可选每年1次每6个月

5.3 疗效评估

疗效 判定标准
显效BMI下降≥2个标准差单位，或BMI Z评分下降≥0.5
有效BMI下降1-2个标准差单位，或BMI Z评分下降0.25-0.5
稳定体重增长速度减慢，BMI保持稳定或轻度上升
无效BMI持续上升，或无明显改善

5.4 心理健康关注

  • 关注患儿情绪，预防自卑、焦虑、抑郁等心理问题
  • 鼓励患儿积极参与社交活动，避免因体重受到嘲笑
  • 必要时请心理科/精神科会诊
  • 强调健康比体重更重要，培养积极的身体意象

5.5 长期管理目标

🎯 长期目标：
• 短期（1-3个月）：建立健康生活方式，控制体重增长
• 中期（3-12个月）：BMI逐渐改善，养成运动习惯
• 长期（1-5年）：维持健康体重，预防并发症，终身健康生活方式
