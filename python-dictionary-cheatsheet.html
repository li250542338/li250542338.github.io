🐍 Python 字典学习总结

Dictionary - 键值对的魔法世界

📖 什么是字典？

字典是 Python 中一种无序、可变的键值对（key-value pairs）集合。每个键都唯一对应一个值，就像现实生活中的字典一样——用“词”（键）查“义”（值）。

💡 核心特点

  • 无序：在 Python 3.7+ 中字典会保持插入顺序
  • 可变：可以随时添加、删除、修改元素
  • 键唯一：同一个键只能出现一次
  • 键不可变：键必须是不可变类型（字符串、数字、元组等）

🛠️ 基本操作

1. 创建字典

# 方式一：使用花括号 {} person = { "name": "张三", "age": 25, "city": "北京" } # 方式二：使用 dict() 函数 person = dict(name="张三", age=25, city="北京") # 方式三：空字典 empty_dict = {} empty_dict = dict()

2. 访问元素

person = {"name": "张三", "age": 25} # 方式一：使用 [] （键不存在会报错） print(person["name"]) # 输出: 张三 # 方式二：使用 get() 方法（键不存在返回 None 或默认值） print(person.get("age")) # 输出: 25 print(person.get("gender", "未知")) # 输出: 未知

3. 添加和修改元素

person = {"name": "张三"} # 添加新键值对 person["age"] = 25 person["city"] = "北京" # 修改已有键的值 person["age"] = 26 # 批量更新 person.update({"gender": "男", "job": "医生"}) print(person) # 输出: {'name': '张三', 'age': 26, 'city': '北京', 'gender': '男', 'job': '医生'}

4. 删除元素

person = {"name": "张三", "age": 25, "city": "北京"} # 删除指定键 del person["city"] # 删除并返回值 age = person.pop("age") # 清空字典 person.clear() print(person) # 输出: {}

🔍 常用方法

📋 keys()

获取所有键

person.keys()

📝 values()

获取所有值

person.values()

🔗 items()

获取所有键值对

person.items()

遍历字典

person = {"name": "张三", "age": 25, "city": "北京"} # 遍历键 for key in person: print(key) # 遍历值 for value in person.values(): print(value) # 遍历键值对（最常用） for key, value in person.items(): print(f"{key}: {value}")

✨ 进阶技巧

1. 字典推导式

# 创建一个平方数字典 squares = {x: x**2 for x in range(1, 6)} print(squares) # 输出: {1: 1, 2: 4, 3: 9, 4: 16, 5: 25} # 过滤字典 scores = {"张三": 85, "李四": 92, "王五": 78} good_scores = {k: v for k, v in scores.items() if v >= 80} print(good_scores) # 输出: {'张三': 85, '李四': 92}

2. 嵌套字典

# 字典中可以再嵌套字典 hospital = { "内科": { "主任": "王医生", "床位": 50, "医生": ["张医生", "李医生"] }, "外科": { "主任": "赵医生", "床位": 35 } } print(hospital["内科"]["主任"]) # 输出: 王医生

🏥 医院质控办实战示例

# 用字典管理科室质控数据 quality_control = { "2024-04": { "内科": {"合格率": 98.5, "病历数": 120}, "外科": {"合格率": 97.2, "病历数": 85} } } # 查询内科合格率 rate = quality_control["2024-04"]["内科"]["合格率"] print(f"内科合格率: {rate}%")

⚠️ 常见陷阱

❌ 键不存在报错

使用 [] 访问不存在的键会抛出 KeyError，建议用 get()

❌ 可变类型作为键

列表不能作为键，但元组可以！

❌ 遍历中删除

不要在遍历字典时直接删除元素，会出问题！
